# Task 1: Business Sales Analysis using Jupyter Notebook

## Objective
To analyze e-commerce transaction data and identify sales trends, best-selling products, and high-revenue categories.

## Tools Used
- Python
- Jupyter Notebook
- Pandas
- Matplotlib
- Google Colab

## Key Insights
- Electronics category generated the highest sales.
- Sales showed an increasing trend across months.
- Laptop and Smartphone are the top-selling products.
- South region contributed the highest revenue.
